## SCREENSHOT REQUIREMENTS

- **Minimum:** 3 screenshots per device
- **Maximum:** 10 screenshots per device
- **Format:** PNG or JPEG
- **No alpha channel** (no transparency)
- **Resolution:** 72 dpi minimum
- **Status bar:** Optional (can include or exclude)

## DESIGN TEMPLATE

### Layout Structure
```
┌────────────────────────┐
│   Top Text Caption     │  ← 15% height
│   (App Feature)        │
├────────────────────────┤
│                        │
│                        │
│     Device Mockup      │  ← 70% height
│     with Screenshot    │
│                        │
│                        │
├────────────────────────┤
│   Bottom Text/Badge    │  ← 15% height
│   (Optional)           │
└────────────────────────┘
```

### Color Scheme
- **Background:** #0A0E1A (dark) or gradient
- **Text:** #FFFFFF (white)
- **Accent:** #00D4FF (cyan) or #8B7FFF (purple)

### Typography
- **Headlines:** Orbitron Bold, 48-64px
- **Subheadlines:** Space Grotesk Medium, 24-32px
- **Max 4 words** per headline

---

## DEVICE FRAME OPTIONS

### Option 1: With Device Frame
- Use official Apple device frames
- Frame color: Space Black or Silver
- Slight drop shadow

### Option 2: Edge-to-Edge (No Frame)
- Full-bleed screenshot
- Rounded corners to match device
- Works well for gameplay shots

### Option 3: Floating Mockup
- Device at slight angle (15-30°)
- Drop shadow for depth
- Good for feature highlights

## SCREENSHOT FLOW (STORY)

Tell a story with your screenshots:

1. **Hook** - "60 Brain Games" (impressive number)
2. **Experience** - Show actual gameplay
3. **Progress** - Mastery and achievements
4. **Personalization** - Theme options
5. **Social** - Multiplayer (if applicable)

---

## DO's ✅

- Use actual in-app screenshots
- Match screenshot content to captions
- Keep text minimal and readable
- Use consistent style across all screenshots
- Localize text for each language
- Test readability at small sizes

## DON'Ts ❌

- Don't use placeholder/fake content
- Don't overcrowd with text
- Don't use low-quality images
- Don't include offensive content
- Don't show pricing (may change)
- Don't show competitor apps
