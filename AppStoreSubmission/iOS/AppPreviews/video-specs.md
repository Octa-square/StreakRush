# iOS APP PREVIEW VIDEO SPECIFICATIONS

## Required Resolutions

| Device Display | Resolution | Aspect Ratio |
|----------------|------------|--------------|
| 6.7" iPhone 14 Pro Max | 1290 × 2796 | 9:19.5 |
| 6.5" iPhone 11 Pro Max | 1242 × 2688 | 9:19.5 |
| 5.5" iPhone 8 Plus | 1242 × 2208 | 9:16 |
| 12.9" iPad Pro | 2048 × 2732 | 3:4 |

## Technical Requirements

| Property | Requirement |
|----------|-------------|
| Format | M4V, MP4, or MOV |
| Codec | H.264 |
| Audio | AAC stereo, 256kbps |
| Duration | 15-30 seconds |
| Frame Rate | 30 fps |
| Bit Rate | 10-20 Mbps |

## Content Guidelines

### Do's ✅
- Show actual app functionality
- Demonstrate key features
- Use in-app audio/music
- Include captions for accessibility
- Show value proposition quickly

### Don'ts ❌
- No device hands/fingers
- No "tap here" overlays
- No prices (may change by region)
- No Apple hardware imagery
- No competitor mentions

## Recording Tips

1. Use iPhone's built-in screen recorder
2. Disable notifications before recording
3. Use Do Not Disturb mode
4. Plan interactions before recording
5. Record at native resolution
6. Keep battery above 80% (no charging icon)

## Poster Frame

- First frame shown before video plays
- Should be compelling and representative
- Usually shows app logo or key UI
- Set manually in App Store Connect
