## LOCAL DATA STORAGE

### What We Store Locally

| Data | Purpose | Location | User Control |
|------|---------|----------|--------------|
| Display Name | Personalization | localStorage | Editable/Deletable |
| Game Scores | Progress tracking | localStorage | Deletable |
| Mastery Tiers | Achievement display | localStorage | Deletable |
| Brain Score | Progress metric | localStorage | Deletable |
| Theme Preference | UI customization | localStorage | Editable |
| Sound Settings | User preference | localStorage | Editable |
| Language | Localization | localStorage | Editable |

### Data Retention
- All data is retained locally until user deletes it
- Uninstalling the app removes all data
- No cloud backup (unless device-level backup)
- No server-side storage

### User Data Rights
Users can:
1. **View** - See all their data in the app
2. **Edit** - Change display name, preferences
3. **Delete** - Reset All Progress in Settings
4. **Export** - Via device backup (iOS/Android)
