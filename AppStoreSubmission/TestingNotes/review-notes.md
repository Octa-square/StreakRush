## DEMO / TESTING INSTRUCTIONS

### No Account Required
CogniXis does not require account creation or login. Reviewers can start using the app immediately after installation.

### Testing Premium Features

To test premium features without purchase:

1. Open the app
2. Tap the **Settings** button (⚙️ gear icon in top-right)
3. Scroll down to **Developer Mode** section
4. Tap **"Enable Premium (Demo)"**
5. All 60 games and premium features are now unlocked

**Note:** This demo toggle is for testing purposes and will be removed before production deployment.

### Resetting Progress
To start fresh:
1. Open Settings (⚙️)
2. Scroll to **Progress** section
3. Tap **"Reset All Progress"**
4. Confirm the reset

## FEATURE WALKTHROUGH

### Home Screen
- **Brain Score Badge** - Shows current game status
- **Progress Bar** - Games completed out of total
- **Next Challenge Card** - System-selected next game
- **Games Grid** - Tap any game to play
- **Load More** - Shows additional games (12 at a time)

### Playing a Game
1. Tap a game from the grid or "BEGIN CHALLENGE"
2. View the **Instructions Screen** with game preview
3. Tap **"I'M READY!"** to start
4. 3-2-1 countdown begins
5. Play the 60-second game
6. View results and mastery tier

### Settings
- **Sound** - Toggle sound effects
- **Vibration** - Toggle haptic feedback
- **Theme** - Choose from 4 color themes
- **Language** - English, Spanish, French
- **Difficulty** - Normal (60s), Hard (45s), Extreme (30s)

### Party Mode (Multiplayer)
1. Tap **Party** in bottom navigation
2. **Join Room** - Enter 6-character room code
3. **Host Room** - Create a room (requires Party Host purchase)
4. Wait for players (2-8)
5. Play 5 games together
6. View final standings

## TROUBLESHOOTING

### If App Won't Load
1. Force close and reopen
2. Clear app cache (Android)
3. Reinstall the app

### If Games Not Responding
1. Ensure touch is on game area
2. Check if game requires specific input (tap vs swipe)
3. Wait for countdown to complete before interacting

### If Purchase Fails
1. Check internet connection
2. Verify payment method in App Store/Play Store
3. Try again later
4. Contact support@cognixis.app
