### Pronunciation
kog-NEEK-sis (emphasis on second syllable)

### Tagline
**"Feed Your Brain"**

Alternative taglines:
- "Train Your Mind Daily"

## LOGO

### Primary Logo
The CogniXis logo consists of:
1. **Brain Icon** - Stylized brain with neural connections
2. **Wordmark** - "CogniXis" in Orbitron font

### Logo Variations
| Version | Usage |
|---------|-------|
| Full Logo (Icon + Text) | Primary, marketing materials |
| Icon Only | App icon, favicons, small spaces |
| Wordmark Only | Text-heavy contexts |
| Horizontal | Wide spaces, headers |
| Stacked | Square spaces, social media |

### Clear Space
Minimum clear space around logo = height of "X" in CogniXis

### Minimum Size
| Format | Minimum Width |
|--------|---------------|
| Print | 1 inch / 25mm |
| Digital | 100 pixels |
| Icon | 32 pixels |

### Logo Don'ts
- ❌ Don't stretch or distort
- ❌ Don't change colors arbitrarily
- ❌ Don't add effects (shadows, bevels)
- ❌ Don't place on busy backgrounds
- ❌ Don't rotate
- ❌ Don't rearrange elements

## TYPOGRAPHY

### Primary Typeface: Orbitron
**Usage:** Headlines, logo, emphasis

| Weight | Usage |
|--------|-------|
| Bold (700) | Headlines |
| Black (900) | Logo, major emphasis |

### Secondary Typeface: Space Grotesk
**Usage:** Body text, UI elements

| Weight | Usage |
|--------|-------|
| Light (300) | Subtle text |
| Regular (400) | Body text |
| Medium (500) | UI labels |
| Semi-Bold (600) | Emphasis |
| Bold (700) | Strong emphasis |

### Type Scale

| Element | Font | Size | Weight |
|---------|------|------|--------|
| H1 | Orbitron | 32px | Bold |
| H2 | Orbitron | 24px | Bold |
| H3 | Space Grotesk | 20px | Semi-Bold |
| Body | Space Grotesk | 16px | Regular |
| Caption | Space Grotesk | 14px | Regular |
| Small | Space Grotesk | 12px | Regular |

### Line Heights
- Headlines: 1.2
- Body: 1.5
- UI: 1.4

## ICONOGRAPHY

### Icon Style
- Stroke-based (not filled)
- 2px stroke weight
- Rounded caps and joins
- Consistent visual weight

### Icon Sizes

| Context | Size |
|---------|------|
| Navigation | 24px |
| In-content | 20px |
| Buttons | 16px |
| Badges | 14px |

### Icon Colors
- Primary: White (#FFFFFF)
- Accent: Cyan (#00D4FF) or Purple (#8B7FFF)
- Disabled: White 40%

## IMAGERY

### Style
- Dark, moody backgrounds
- Glowing/neon accents
- Abstract neural patterns
- Minimal, focused compositions

### Themes
- Brain and cognition
- Neural networks

## UI COMPONENTS

### Buttons

| Type | Style |
|------|-------|
| Primary | Cyan background, white text |
| Secondary | Transparent, cyan border |
| Accent | Gradient background |
| Disabled | 40% opacity |

### Cards
- Background: rgba(139, 127, 255, 0.08)
- Border: 1px solid rgba(255, 255, 255, 0.1)
- Border-radius: 16px
- Padding: 20px

### Input Fields
- Background: rgba(255, 255, 255, 0.05)
- Border: 1px solid rgba(255, 255, 255, 0.2)
- Border-radius: 12px
- Focus: Cyan border

## MOTION

### Timing
| Type | Duration | Easing |
|------|----------|--------|

## VOICE & TONE

### Brand Personality
- Intelligent but approachable
- Energetic but not hyper
- Modern and tech-forward
- Encouraging and positive

### Writing Style
| Do | Don't |
|-----|-------|

### Messaging Themes
- Quick and efficient
- Fun and engaging
- Progress and growth
- Challenge and achievement
